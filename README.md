# Kamailio SIP Server - Credits

Kamailio SIP Server project appreciates the efforts made by anybody to improve its quality or to promote it,
no matter it is via code, documentation, security analysis, online advocacy or participation to events.

While contributions to code and some documentation components could be referenced via git commit history,
other types of works are not implicitely visible.

This repository enables contributors to list their name and scope of work by making Pull Requests
to update the content.

## Conditions

To get a pull request approved, the submitters have to provide proofs of their work for Kamailio project.
If the contribution was acknowledged privately by one of the Kamailio administrative groups (e.g., management, security),
the pull requests will be aproved without requesting other details, but if the work was done independenly,
without prior discussions with one of the Kamailio administrative groups, then the description of the pull
request has to include details about the type of work, when was done and what were the results.

Make the pull request to update the list of credits in alphabetical order.
